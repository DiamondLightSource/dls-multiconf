Naming conventions
=======================================================================

Here are the naming conventions used within the source code.

variables
    lowercase, underscores

constants
    uppercase, underscores

classes
    camel case

class methods
    lowercase, underscores

imports

    exception: top-level packages __init__.py will have __version__ defined in them

python packages
    lowercase, underscores

repository
    lowercase, hyphens
